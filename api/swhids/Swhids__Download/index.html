val any : Swhids.Lang.identifier -> ((string, string) Stdlib.result list, string) Stdlib.Result.t

For any object identifier, compute a list of URLs from which the object can be downloaded. For all kind of object, the list should contain a signe URL except for snapshot objects which may lead to a list of many URLs (one URL per branch).
